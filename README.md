# Quiz-App
A Fun, interactive quiz app that tests your knowledge of general knowledge. Built with HTML, CSS, and Vanilla JavaScript.
## Live Demo 
[view the live app](https://quizapp-trent.netlify.app)
## Features
- 5 multiple-choice questions (General Knowledge)
- Real-time score tracking
- visual feedback: correct answer turns green, wrong answer turns red (and correct one is highlighted)
- Answer locking- user can only select one answer per question
- Next button to move through questions
- Final  score screen with option to restart the quiz
- fully responsive design (works on mobile and desktops)
## Technoligies used
HTML5
CSS3 (red, white, and blue theme)
JavaScript (ES6)
Netlify for hosting
